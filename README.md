

المهمة الأولى - المخزن الدائري بلغة C

الفكرة
هذا البرنامج يوضح كيفية استخدام المخزن الدائري في لغة C لتخزين وقراءة البيانات.

ما يقوم به البرنامج
- يطلب من المستخدم إدخال اسمه.  
- يضيف النص "CE-ESY" إلى الاسم.  
- يخزن النتيجة داخل المخزن الدائري حرفًا حرفًا.  
- يقرأ البيانات من المخزن ويعرضها على الشاشة.  
- يتأكد في النهاية أن المخزن أصبح فارغًا.  

مميزات المخزن
- التهيئة.  
- الكتابة.  
- القراءة.  
- التعامل مع حالة الامتلاء (Overflow).  
- التعامل مع حالة الفراغ (Underflow).
